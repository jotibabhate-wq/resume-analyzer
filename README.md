# AI Resume Analyzer

A full-stack AI-powered resume analyzer built with HTML, CSS, JavaScript, Node.js, MongoDB, and Python.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- AI Engine: Python (NLP)

## Setup

### 1. Install Node dependencies
npm install

### 2. Install Python dependencies
cd ai_engine
pip install -r requirements.txt

### 3. Start the server
npm run dev

### 4. Open in browser
http://localhost:5000
